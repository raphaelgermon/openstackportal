OpenStack Management Portal

A unified, vCenter-like experience for multi-cluster OpenStack management.

This portal is designed to bridge the gap between traditional virtualization management and modern cloud infrastructure. It provides a centralized, hierarchical view of global infrastructure across multiple Availability Zones, Clusters, and Physical Hosts.

Key Features

Hierarchical Inventory
Drill-down from Region to Cluster to Host to Instance.
Real-time Analytics
Live capacity tracking for CPU and RAM across all regions.
No-Auth Console
One-click web console access to any instance via proxy.
Unified Alerting
Hardware and Software health checks in a single view.
Maintenance Mode
Host evacuation and maintenance scheduling.
Audit Logging
Comprehensive logs for user actions and system events.
